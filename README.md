# 代号X1

Godot 4 游戏项目。

## 环境要求

- [Godot 4.2+](https://godotengine.org/download)（推荐 4.4）

## 打开项目

1. 启动 Godot Engine
2. 选择 **Import** 或 **打开**
3. 选中本目录下的 `project.godot`

或在命令行（需将 Godot 加入 PATH）：

```powershell
godot --path "C:\Users\zhipengjiao\Projects\代号X1"
```

## 目录结构

```text
代号X1/
├── project.godot      # 项目配置
├── icon.svg           # 项目图标
├── scenes/main/       # 主场景
├── scripts/           # GDScript 脚本
└── assets/            # 美术、音频等资源
```

## 主场景

- 入口：`scenes/main/main.tscn`
- 脚本：`scripts/main.gd`
